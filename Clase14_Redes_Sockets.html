Resumen TCP/IP


Direcciones IP:

  • IPv4 = 232
  • IPv6 = 2128


Puertos:

  • TCP = 216
  • UDP = 216


Cada uno, espacios de puertos por protocolo.

Ejemplos de IPs

  • Ejemplo red IP:
    • Red: 192.168.0.0
    • Mascara v1: 255.255.255.0
    • Mascara v2: /24
    • Router: 192.168.0.1
    • Broadcast: 192.168.0.255
    • Host1: 192.168.0.10
    • Host2: 192.168.0.11
    • Host3: 192.168.0.12
    • Dir loopback: 127.0.0.1
