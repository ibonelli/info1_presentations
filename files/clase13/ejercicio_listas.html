Ej. Listas

Realizar un programa que permita manejar los alumnos de un curso.

El programa deberá presentar el siguiente menu:

  1. Ingresar un Alumno
  2. Buscar un Alumno
  3. Borrar un Alumno
  4. Dar Promedios
  5. Salir

La estructura del Alumno será la siguiente:

  • Nombre: string 40 caracteres
  • Apellido: string 40 caracteres
  • Legajo: Numero que puede ser mayor a 999999 (long)
  • 3 Notas (1p, 2p, proyecto): 0 a 10 (float)

Dar promedios generará el siguiente reporte:

  • Nombre: string 40 caracteres
  • X Aprobados (1p >4 && 2p >4 && proy >4)
  • X A recuperatorio
  • Promedio de aprobación: Sum[(1p+2p+proy)/3]/#aprob

Para hacer mas sencilla la carga de datos y pruebas del programa se pide:

  • Al comienzo cargar los datos del archivo alumnos.dat
  • Al finalizar guardar los datos en memoria en el mismo archivo

Si el archivo es binario solo lo podrá crear usando este programa, si en su lugar lo hace modo texto lo podrá cargar en forma externa.

Se pide resolver este ejercicio usando implementando una lista simplemente enlazada.

Nota 1: La lista estará ordenada por Legajo
Nota 2: El manejo (ingreso, busqueda, etc...) se hará por Legajo
