Info1 - Linux

Linux año a año...

1993 a 1995

  • 1993: Más de 100 desarrolladores trabajan sobre el núcleo Linux. En este año también el proyecto de Wine comienza su desarrollo. También, la distribución más antigua actualmente activa, Slackware, es liberada por primera vez. Más tarde en el mismo año, el Proyecto Debian es establecido.
  • 1994: Se libera la versión 1.0 de Linux. Esta versión está, por primera vez, disponible en la red. El proyecto XFree86 contribuye con una interfaz gráfica de usuario (GUI). En este año las empresas Red Hat y SUSE publican la versión 1.0.
  • 1995: Kernel v1.2. Linux es portado a DEC y a SUN SPARC. Durante los años siguientes es portado a un número cada vez mayor de plataformas.

1993 a 1999

  • 1996: La versión 2.0 del núcleo Linux es liberada. El mayor cambio es que se hace multiprocesador transformándose en una alternativa muy atractiva para empresas.
  • 1997: Varios programas propietarios son liberados para Linux: Adabas D, Netscape y StarOffice (entre otros).
  • 1998: IBM, Compaq y Oracle (entre otras) anuncian soporte para Linux. Se comienza a desarrollar el interfaz gráfica de usuario KDE.
  • 1999: Linux v2.2 con soporte a SMP mejorado. Comienza GNOME.

2000 a 2005

  • 2000: StarOffice pasa a ser GPL
  • 2001: Linux v2.4 soportando hasta 64 Gb de RAM, sistemas de 64 bits, dispositivos USB y un sistema de archivos journaling.
  • 2002: OpenOffice.org libera la versión 1.0 de su Suite de oficina. Mozilla es también liberado.
  • 2004: El equipo de XFree86 se desintegra y se forma la fundación X.Org, que provoca un desarrollo considerablemente más rápido del servidor X para Linux.
  • 2005: El proyecto openSUSE es comenzado como una distribución libre de la comunidad de Novell.

2006 a hoy

  • 2006: Comienza el uso de efectos acelerados por hardware sobre el escritorio Linux. Oracle publica su propia distribución de Red Hat.
  • 2007: Dell es el primer fabricante principal de computadoras en vender una computadora personal de escritorio con Ubuntu instalado.
  • 2008: Primer teléfono Android (El kernel de Android es basado en el de Linux)
  • 2011: Version 3.0 of the Linux kernel is released
  • 2012: Se lanza al mercado Raspberry Pi (computadora del tamaño de una tarjeta de crédito con ARM y Linux)
